# Kaizen Art İnşaat & Mühendislik

Bu proje, Kaizen Art için hazırlanmış modern, sinematik ve mimari odaklı bir web sitesidir.

## Özellikler
- **Sinematik Tasarım:** Framer Motion ile zengin animasyonlar ve geçişler.
- **Mimari Estetik:** Modern ızgara sistemleri ve yüksek kaliteli görseller.
- **Responsive:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu.
- **SPA:** React ve React Router ile hızlı sayfa geçişleri.

## Teknolojiler
- React
- Tailwind CSS
- Framer Motion
- Lucide React
- Vite

## Kurulum ve Çalıştırma

```bash
# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev

# Üretim için derleyin
npm run build
```
